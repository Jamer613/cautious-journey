Projeto Analise de crédito, utilizando Pycaret para automação e predição com modelos de classificação de Machine Learning.

- Base de dados obtida no repositório do Kaggle.
- linguagem utilizada - Python.
- API - FastAPI e Uvicorn
- Container Docker para execução da API consumindo o modelo em máquina virtual.
- Dashboard com powerBI, para demosntrar desempenho dos dados preditos com porção de teste e validação.
